# 📜 목차
<hr/>

1. 변수
- 변수 선언과 변수 초기화는 필수
- 변수 사용과 변수 사용범위(Scope)
   1. 타입
      - 📌자바에서는 **타입**이 매우 중요
      ➡ 타입이 맞지 않으면 오류 발생
      - 정수 타입: int, long
      - 실수 타입: double
      - 논리 타입: boolean
      - 자동 타입 변환(Promotion) : 작은 범위 -> 큰 범위
      - 강제 타입 변환(Casting) : 큰 범위 -> 작은 범위 (캐스팅 연산자)
   2. 연산자
  - 단항 연산자, 이항 연산자, 삼항 연산자
  - 산술 연산자, 대입 연산자, 증감 연산자 등
  - 연산자 우선순위 존재 ➡ ()안 부터 먼저 연산
2. 조건문
- if
- if - else, if - else if - else
- switch - case
- 향상된 switch 문
3. 반복문
- while
- for(향상된 for문)
- do - while
- break, continue
4. 배열
- 배열 선언, 배열 초기화, 배열 길이
- 다차원 배열
- 향상된 for문과 잘 사용됨
5. 메서드
- 메서드 선언, 메서드 호출
- 리턴값, 리턴 타입
- 매개변수(parameter), 인자(argument)
- 메서드 오버로딩(overloading)
